Espacio de información académica y publicación de material del curso de Base de Datos II. Aquí el docente comparte recursos, lecturas, guías y herramientas organizadas por unidad para el Ciclo V · 2026-I.

🗄️
SQL & PostgreSQL
Motor relacional de la industria: consultas, procedimientos y optimización de bases de datos.
📐
Modelado E-R
Diseño conceptual: entidades, atributos, relaciones y diagramas entidad-relación.
🔄
Normalización
Organización de tablas para eliminar redundancia: 1FN, 2FN, 3FN y BCNF.
🌐
NoSQL
MongoDB, Redis y Cassandra: bases de datos no relacionales para sistemas modernos.
🔒
Seguridad en BD
Gestión de usuarios, roles, cifrado y protección contra SQL injection en producción.
☁️
Cloud & Supabase
Bases de datos en la nube: AWS RDS, Google Cloud SQL y Supabase como servicio.
